02 | Python 开发环境搭建：10 分钟搞定，写出你的第一行代码

上一篇我们聊了 Python 能做什么、零基础该怎么学，很多同学已经摩拳擦掌，想亲手写代码试试了。

学编程的第一步，永远是先把「干活的工具」准备好。很多新手还没开始写代码，就倒在了环境搭建这一步：要么下载错了版本，要么装完运行不了，折腾半天没搞定，直接劝退。

别担心，今天这篇我会用最直白的步骤，带你从零搭好 Python 开发环境。全程跟着点鼠标就行，哪怕你完全不懂电脑，也能 10 分钟搞定，顺利写出你的第一行 Hello World。

一、Python 解释器下载与安装

电脑本身只认识 0 和 1，看不懂我们写的 Python 代码。 解释器的作用，就是把我们写的 Python 代码，翻译成电脑能听懂的指令，让代码真正跑起来。

简单说：没有解释器，你写的代码就是一堆纯文本，根本运行不了。

我们先装最核心的解释器，分 Windows 和 Mac 两个系统来讲，照着步骤点就行。

👉 Windows 系统安装步骤

1.1 官网下载安装包

打开 Python 官方下载页：https://www.python.org/downloads/windows/

往下翻，找到「Stable Releases」（稳定版本），推荐下载 Python 3.11.x 或 3.12.x 的稳定版（不用追求最新的测试版，稳定版兼容性最好）。

点击对应的「Windows installer (64-bit)」下载，现在的电脑基本都是 64 位系统，直接下这个就行。

1.2 安装：这一步千万注意！

下载完成后，双击打开安装包。

⚠️ 重中之重：一定要勾选最下方的「Add Python.exe to PATH」！

这是新手最容易踩的坑！不勾选的话，后续电脑找不到 Python，输命令会报错，新手很难自己解决。

勾好之后，直接点「Install Now」默认安装就行，它会自动装到 C 盘，不用改路径，省得麻烦。

等进度条走完，出现「Setup was successful」就说明安装成功了，点 Close 关掉就行。

1.3 验证：确认装好了

按键盘 Win + R 键，弹出运行框，输入 cmd 回车，打开黑色的「命令提示符」窗口。

如果忘记勾选了，也有补救措施——配置环境变量

右键–>此电脑–>属性–>高级系统设置–>环境变量，在系统变量中找到 Path，点击编辑，新建，添加 python 的安装路径，完成后点击确定，重启命令行窗口，再重试是否安装成功。

👉 Mac 系统安装步骤

Mac 电脑自带 Python 2.x 版本，但这个版本早就淘汰了，我们必须装 Python 3。

方法一：官网安装包（新手推荐，最简单）

  1. 打开官网下载页：https://www.python.org/downloads/macos/
  2. 找到稳定版（推荐 3.11.x/ 3.12.x），点击「macOS 64-bit universal2 installer」下载。
  3. 下载完成后双击 pkg 安装包，一路点「继续」「安装」，输入电脑密码确认即可。

方法二：Homebrew 安装（有一定基础可选）

新手直接用第一种安装包方式就行，简单不容易出错。

二、安装 PyCharm

PyCharm 是专门做 Python 开发的 IDE，功能更专业，也是目前企业里使用最多的 Python 开发工具。

2.1 版本说明

PyCharm 分为两个版本：

  • 社区版（Community）：完全免费，核心功能齐全，新手学习、日常开发完全够用，推荐所有人优先选这个。
  • 专业版（Professional）：收费，多了 Web 开发、数据库集成、远程开发等高级功能，学生可以凭教育邮箱免费申请，新手暂时用不上。

2.2 下载安装

打开 PyCharm 官方下载页：https://www.jetbrains.com/pycharm/download/

网站会自动识别你的操作系统，找到「Community」社区版，点击「Download」下载即可。

Windows 系统安装步骤

下载完成后，双击安装包运行，点击「Next」下一步；

选择安装路径，默认装在 C 盘，也可以自己改到其他盘，注意路径里不要有中文和空格，然后点「Next」；

安装选项页，建议把这几个都勾上：

  • ✅ Create Desktop Shortcut：创建桌面快捷方式
  • ✅ Add “Open Folder as Project”：右键文件夹可以直接用 PyCharm 打开
  • ✅ Add launchers dir to the PATH：添加到系统环境变量
  • .py 关联选项可以按需勾选，勾完点「Next」；

直接点「Install」开始安装，等待进度条走完；

安装完成后，勾选「Run PyCharm Community Edition」，点「Finish」启动软件。

Mac 系统安装步骤

下载完成后打开 dmg 安装包，直接把 PyCharm 图标拖到「应用程序」文件夹里；

打开启动台，找到 PyCharm 点击启动，首次打开会提示是否导入设置，选「Do not import settings」，点 OK 即可。

2.3 启动配置

第一次打开 PyCharm，会进入配置向导，跟着走就行：

  • 选择主题：左边 Darcula 是黑色主题，右边 Light 是白色主题，选自己看着舒服的就行，点「Next」；
  • 特色插件页面，新手直接点「Start using PyCharm」跳过，不用额外安装。

⚠️ 接下来这一步非常关键！PyCharm 本身只是写代码的工具，必须绑定我们之前安装的 Python 解释器，代码才能运行。

我们通过创建第一个项目来完成配置：

  1. 点击欢迎页的「New Project」新建项目；
  2. Location 是项目保存路径，自己选一个纯英文的文件夹，比如 G:\python_project\first_demo；
  3. 重点看「Interpreter Type」选项：
    1. 选择「Project venv」（之前配置好的解释器）；
    2. 找到我们之前安装的 Python 解释器路径;
    3. 选好后点「OK」确认；
  4. 取消勾选「Create a welcome script」，然后点「Create」创建项目。

到这里，PyCharm 的环境就配置完成了。

四、新手避坑：环境搭建最常见的 5个问题

很多人第一次装环境都会踩坑，我把最常见的问题列出来，遇到了直接对照解决。

4.1 输 python 提示「不是内部或外部命令」

  • Windows：99% 是安装时没勾选「Add Python to PATH」。最简单的解决办法：重装一遍安装包，一定要记得勾上。（如果不想重新安装，也可以手动配置环境变量，上面有教程）
  • Mac：自带的是 Python 2，要用 python3 命令，不要直接输 python。

4.2 代码运行报错，提示找不到文件

大概率是文件路径里有中文、空格或者特殊字符，比如你把文件存在了「桌面 / 我的代码」文件夹里。 新手先把代码存在纯英文路径的文件夹里，比如 D:\code\python，避免各种奇奇怪怪的问题。

4.3 文件名命名为 python.py/print.py

绝对不要用 Python 的关键字、命令名当文件名，会和系统本身的功能冲突，导致运行报错。 起名就用简单的英文，比如 hello.py、test.py 就好。

4.4 PyCharm 提示「No Python interpreter configured」

这是新手用 PyCharm 最常见的报错，意思是没配置 Python 解释器。

解决方法：点击 PyCharm 右下角的「No Interpreter」→ 「Add New Interpreter」→「System Interpreter」，选择你安装的 python.exe 路径即可。

4.5 PyCharm 创建项目后运行特别慢

如果选了「New environment using Virtualenv」创建虚拟环境，首次加载会慢一些。新手入门阶段直接用系统解释器就行，不用创建虚拟环境，后面进阶篇我们会讲虚拟环境的用法。

五、今天的动手练习

学编程光看不练等于白学，今天的任务很简单，务必亲手完成：

  1. 按照步骤装好 Python 解释器，PyCharm 代码编辑器安装配置完成，验证版本号正常；
  2. 用脚本文件方式，成功运行出 Hello World；
  3. 修改代码，让程序输出你的名字、年龄，再试试用 input() 做个简单的问答。

完成这三步，你就已经正式跨进 Python 的大门了。
